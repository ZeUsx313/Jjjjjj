# ChatZEUS (شات زيوس) 🤖

تطبيق ويب شخصي يعمل بالكامل على المتصفح (Client-Side Only)، يحاكي بيئة محادثة متعددة الوكلاء (Multi-Agent). يمكنك إنشاء وكلاء ذكاء اصطناعي، وإعطاؤهم شخصيات وأدوار، ثم توكيلهم بمهمة. عند تحديد مهمة، يتواصل الوكلاء فيما بينهم بشكل تلقائي حتى يصلوا إلى نتيجة نهائية.

## ✨ المميزات

### 🔧 إنشاء الوكلاء (Agents)
- إنشاء عدة وكلاء كل منهم له:
  - اسم مخصص
  - صورة رمزية (Avatar) ثابتة أو افتراضية
  - شخصية أو أسلوب تواصل (Role/Persona)
  - نموذج ذكاء اصطناعي معين من مزود محدد
  - صلاحيات أو مجال تخصص (برمجة، مراجعة، تحليل…)

### 🎭 الوسيط (Orchestrator)
- عند إدخال مهمة من المالك:
  - الوسيط يوزع المهمة على الوكلاء
  - يبدأ الوكلاء بالتواصل التلقائي فيما بينهم
  - يتم عرض هذا النقاش على الشاشة بشكل رسائل متسلسلة
  - في النهاية يظهر ملخص النتيجة النهائية

### 🔑 إدارة المفاتيح (API Keys Manager)
- تخزين عدد غير محدود من المفاتيح لكل مزود في ذاكرة المتصفح
- التوزيع الدائري (Round Robin) لتوزيع الطلبات بالتساوي
- التبديل التلقائي (Failover) عند انتهاء الحصة أو فشل الطلب
- عرض حالة واستخدام كل مفتاح

### 📁 إدارة الملفات
- رفع ملفات فردية أو مجلدات كاملة
- تخزين الملفات في ذاكرة المتصفح
- عرض الملفات داخل النظام للكود والنصوص
- ربط الملفات بالمحادثات والمشاريع

### 💬 واجهة المحادثة
- واجهة مشابهة لـ ChatGPT
- شاشة رئيسية للمحادثة (رسائل تظهر بشكل فقاعات chat)
- شريط جانبي (Sidebar) لعرض المحادثات
- زر إعدادات في الأعلى يمين يعرض قائمة الوكلاء والإعدادات والمفاتيح
- كل وكيل له لون مميز أو Avatar بجانب رسائله
- دعم الوضع الليلي والنهاري
- دعم عرض الكود (Syntax Highlighting)

## 🚀 كيفية التشغيل

### الطريقة الأولى: فتح الملف مباشرة
1. قم بتحميل الملفات الثلاثة:
   - `index.html`
   - `style.css`
   - `app.js`
2. افتح ملف `index.html` في المتصفح

### الطريقة الثانية: استخدام خادم محلي
```bash
# باستخدام Python
python3 -m http.server 8000

# باستخدام Node.js
npx http-server

# باستخدام PHP
php -S localhost:8000
```

ثم افتح المتصفح على `http://localhost:8000`

## 📱 كيفية الاستخدام

### 1. إنشاء وكلاء جدد
1. اضغط على زر الإعدادات (⚙️) في الشريط الجانبي
2. اختر تبويب "الوكلاء"
3. اضغط على "إضافة وكيل"
4. املأ البيانات المطلوبة:
   - اسم الوكيل
   - الدور/التخصص
   - الشخصية
   - نموذج AI
5. اضغط "حفظ"

### 2. إضافة مفاتيح API
1. في تبويب "المفاتيح"
2. اضغط على "إضافة مفتاح"
3. اختر المزود (OpenAI, Anthropic, Google)
4. أدخل مفتاح API
5. اضغط "حفظ"

### 3. بدء محادثة جديدة
1. اضغط على "محادثة جديدة" في الشريط الجانبي
2. اكتب مهمتك في صندوق الإدخال
3. اضغط Enter أو زر الإرسال
4. شاهد الوكلاء يتناقشون تلقائياً لحل المهمة

### 4. تبديل الوضع الليلي/النهاري
- اضغط على زر القمر/الشمس في أعلى منطقة المحادثة

## 💾 التخزين

- **localStorage**: للوكلاء، المحادثات، المفاتيح، الملفات النصية
- **sessionStorage**: للمحادثة الجارية (اختياري)
- لا يوجد خادم أو قاعدة بيانات خارجية
- كل البيانات تُخزَّن محلياً في المتصفح

## 🎨 التصميم

- واجهة عربية بالكامل مع دعم RTL
- تصميم حديث وجميل
- دعم الوضع الليلي والنهاري
- تصميم متجاوب (Responsive) يعمل على جميع الأجهزة
- رسوم متحركة سلسة
- ألوان متناسقة ومريحة للعين

## 🔧 التقنيات المستخدمة

- **HTML5**: الهيكل الأساسي
- **CSS3**: التصميم والأنيميشن
- **JavaScript ES6+**: المنطق والتفاعل
- **Prism.js**: تمييز الكود (Syntax Highlighting)
- **Font Awesome**: الأيقونات
- **localStorage**: التخزين المحلي
